Features

Everything you need to ship

No vendor lock-in. Works with your stack.

📝

Auto-Planning

Cipher reads your docs and creates structured stories with acceptance criteria. No manual PRD writing.

🔄

Loop Until Done

Fresh context each iteration. Never loses track. Runs autonomously until all stories complete.

🧪

Test Generation

Cortex writes tests for edge cases: null inputs, error handling, boundary conditions.

🔒

Code Review

Sentinel checks security, patterns, and quality. Approves or creates fix tasks.

🌏

Any AI Tool

Works with Claude, AMP, Cursor, Aider. Not locked to one provider or API.

📦

Linear or JSON

Store stories in Linear for teams or local JSON for solo projects. Simple config switch.
