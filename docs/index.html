Features

Everything you need to ship

No vendor lock-in. Works with your stack.

Auto-Planning

Cipher reads your docs and creates structured stories with acceptance criteria.

Loop Until Done

Fresh context each iteration. Never loses track. Runs until complete.

Test Generation

Cortex writes tests for edge cases, error handling, and boundaries.

Code Review

Sentinel checks security, patterns, and quality before merge.

Any AI Tool

Works with Claude, AMP, Cursor, Aider. No vendor lock-in.

Linear or JSON

Store stories in Linear for teams or local JSON for solo projects.
